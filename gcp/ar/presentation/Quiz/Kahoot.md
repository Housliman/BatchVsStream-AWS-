# اختبار Kahoot! نسخة Google Cloud

1.  **سؤال:** أي نموذج معالجة هو الأفضل لإنشاء تقارير مبيعات نهاية اليوم؟
    *   **الإجابة:** المعالجة بالدُفعات
    *   **الخيارات:** المعالجة بالبث، معالجة المعاملات، المعالجة الهجينة.

2.  **سؤال:** كشف الاحتيال على بطاقات الائتمان في الوقت الفعلي هو حالة استخدام كلاسيكية لـ...
    *   **الإجابة:** المعالجة بالبث
    *   **الخيارات:** المعالجة بالدُفعات، معالجة ETL، معالجة التقارير.

3.  **سؤال:** أي خدمة من خدمات Google Cloud مصممة بشكل أساسي لاستيعاب تدفقات البيانات في الوقت الفعلي؟
    *   **الإجابة:** Pub/Sub
    *   **الخيارات:** Google Cloud Batch, Cloud Storage, BigQuery.

4.  **سؤال:** "زمن استجابة عالٍ" و "مجموعات بيانات كبيرة ومحدودة" هي خصائص أي نموذج؟
    *   **الإجابة:** المعالجة بالدُفعات
    *   **الخيارات:** المعالجة بالبث، المعالجة في الوقت الفعلي، معالجة الأحداث.

5.  **سؤال:** أي خدمة من خدمات Google Cloud هي مستودع بيانات بحجم بيتابايت؟
    *   **الإجابة:** BigQuery
    *   **الخيارات:** Dataproc, Cloud Data Fusion, Pub/Sub.
